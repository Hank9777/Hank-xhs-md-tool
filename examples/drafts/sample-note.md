# 用 Markdown 生成小红书图文|示例封面标题

## 为什么需要自动排版

### 把写作和视觉样式分开

Markdown 适合写结构化内容，模板负责把标题、正文、列表和金句统一排成适合发布的卡片。

> 先稳定内容结构，再稳定视觉样式，批量生产才会轻松很多。

#### 一个简单流程

1. 写好 Markdown。
2. 在网页里选择模板和预设。
3. 调整标题、正文、颜色和间距。
4. 导出 PNG。

---

## 可以继续扩展模板

- 新增模板目录即可出现在模板选择器中。
- `editor.json` 控制左侧可视化参数。
- `renderer.js` 控制 Markdown 语义如何变成页面 HTML。
