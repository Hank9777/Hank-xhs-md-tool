# 小红书|图文排版工具|专注写作2|一键成图！


封面页使用 Markdown 的一级标题 `#` 生成，用 `|` 分隔可以让标题分成多行显示，增加层次感。

封面编辑器可以控制：**主标题文字大小**、**标题颜色**、**强调文字颜色**

> 一个好的封面：标题简洁，颜色对比，排版留白。

---


## 这个工具能帮你做什么？

### 把 Markdown 变成精美小红书卡片

你只需要用 Markdown 写内容，工具会自动帮你排版成 900 × 1200 的小红书图文卡片，支持一键导出高清 PNG。

**适合这些场景：**

- 知识笔记整理和分享
- 方法论文章图文化
- 课程内容可视化输出
- 案例拆解和复盘
- 长文拆分成多页卡片

> 写作和排版分离，让内容创作者专注在内容本身。

## 核心功能一览

### 实时预览和编辑

左侧调整模板参数，右侧实时看到排版效果，所见即所得。

### 智能分页

内容过长会自动分页，保持每一页的阅读舒适度，无需手动切割。

### 模板和预设系统

- 内置多种视觉风格模板
- 可保存常用的样式预设
- 支持自定义新增模板

### 灵活的 Markdown 语法

**支持的元素：**

1. 封面标题（自动分行）
2. 多级标题（H1/H2/H3）
3. 正文段落和强调块
4. 有序列表和无序列表
5. 金句引用块
6. 强制分页符

## 快速上手

### 三步生成你的第一张卡片

1. **导入 Markdown**：点击"选择本地 MD 文件"
2. **调整样式**：左侧修改颜色、字号、间距
3. **导出图片**：点击"导出 PNG"选择保存位置

### 推荐工作流

**第一步：** 先用纯 Markdown 写好内容大纲和正文。

**第二步：** 导入工具，选择适合内容调性的模板。

**第三步：** 微调视觉参数，保存为预设供下次使用。

**第四步：** 批量导出，直接用于小红书发布。

> 这个流程让你把 80% 的时间花在内容上，20% 的时间花在视觉上。



## 进阶使用

### 自定义模板

如果内置模板不满足需求，你可以：

- 现有模板目录：templates/deep-reading/
- 用AI修改模板框架
- 添加到模板库：templates/
- 在编辑器中调整具体元素样式


**无需修改核心代码，模板完全独立。**

### 样式预设管理

把常用的配色方案、字号组合保存成预设，下次一键应用。

### Markdown 技巧

- 用 `|` 分隔封面标题的多行
- 用 `---` 强制开启新页面
- 整段加粗会渲染成强调块
- 引用块自动渲染为金句样式
